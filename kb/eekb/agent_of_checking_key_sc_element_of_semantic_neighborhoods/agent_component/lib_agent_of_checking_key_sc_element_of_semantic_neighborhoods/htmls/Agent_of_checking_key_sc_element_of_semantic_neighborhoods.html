Задачей агента проверки наличия ключевого sc-элемента у указанной семантической окрестности является нахождение семантических окрестностей, у которых не указан ключевой sc-элемент в соответствующей структуре. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия ключевого sc-элемента у указанной семантической окрестности. Единственным аргументом запроса является структура в которой будет произведена проверка семантические окрестности.
