Задачей агентн проверки понятий, которые не соотнесены с предметной областью является нахождение понятий, которые не соотнесены с предметной областью в соответствующей структуре. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки понятий, которые не соотнесены с предметной областью. Единственным аргументом запроса является структура в которой необходимо проверить понятия.
